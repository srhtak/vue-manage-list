### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/srhtak/vue-manage-list.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```   
3. Compiles and hot-reloads for development
   ```sh
    npm run serve
   ```
